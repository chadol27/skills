# Components: Tooltip

Official docs: `https://picocss.com/docs/tooltip`

Pico enables tooltips with data attributes and no JavaScript.

## Key points

- Add `data-tooltip` to links, inline elements, or buttons.
- Tooltip placement is controlled with `data-placement`.
- Supported placements include the default top position plus `right`, `bottom`, and `left`.

## Syntax

```html
<a href="#" data-tooltip="Tooltip">Link</a>
<em data-tooltip="Tooltip">Inline element</em>
<button data-tooltip="Delete item" data-placement="bottom">Delete</button>
```
