# Components: Progress

Official docs: `https://picocss.com/docs/progress`

Pico styles the native `<progress>` element without needing a custom progress component.

## Key points

- Use `<progress value="..." max="...">` for determinate progress.
- Use `<progress>` with no value for the indeterminate state.
- The component stays fully native and semantic.

## Syntax

```html
<progress value="40" max="100"></progress>
<progress></progress>
```
