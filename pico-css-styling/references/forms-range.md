# Forms: Range

Official docs: `https://picocss.com/docs/forms/range`

Pico styles `<input type="range">` as a native slider control.

## Key points

- Use the regular range input syntax.
- Labels remain the preferred wrapper for accessible naming.
- Pico keeps the slider integrated with the rest of the form visual system.

## Syntax

```html
<label>
  Brightness
  <input type="range" />
</label>

<label>
  Contrast
  <input type="range" value="40" />
</label>
```
